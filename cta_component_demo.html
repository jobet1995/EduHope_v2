CTA Component

This is a reusable call-to-action component for the EduHope website.

Join Us in Transforming Education Today

Your support helps provide quality education to children in need. Together, we can build brighter futures.

Donate Now Become a Volunteer

Component Usage

This component can be easily integrated into any page using Django's template include mechanism: 

{% include 'components/cta.html' %}

The component features:

  • Gradient background with image overlay for visual appeal
  • Two prominent CTA buttons with icons (Donate and Volunteer)
  • Fade-in animation when scrolling into view
  • Hover effects with subtle lift and shadow enhancement
  • Responsive design that works on all screen sizes
  • Reusable across different pages
