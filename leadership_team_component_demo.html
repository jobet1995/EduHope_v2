Component Usage

This component can be easily integrated into any page using Django's template include mechanism: 

{% include 'components/leadership_team.html' %}

The component features:

  • Responsive grid layout (1 column on mobile, 2 on tablet, 4 on desktop)
  • Team member cards with profile images and LinkedIn links
  • Fade-in animations on scroll for each team member
  • Hover effects on cards with subtle lift and shadow enhancement
  • Consistent styling with EduHope's color scheme
  • Reusable across different pages
