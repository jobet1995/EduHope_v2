Component Usage

This component can be easily integrated into any page using Django's template include mechanism: 

{% include 'components/impact_statistics.html' %}

The component features:

  • Fully responsive grid layout (1 column on mobile, 2 on tablet, 3 on desktop)
  • Animated count-up statistics that trigger on scroll
  • Gradient background with floating animated elements
  • Hover effects on statistic cards
  • Accessible design with proper contrast
  • Reusable across different pages
